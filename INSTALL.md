# Installation

- Download the **[civil-record package](https://github.com/David-Baron/civil-records/releases)**.
- Unzip the package.
- Upload the unzipped package via FTP/SFTP to the server.
- Open the url of civil-record installer. ex: `https://your-domain.extention/civil-records/install/install.php`
